## [0.1.0]

### Added

- Added abstraction on top of the Rollup HTTP API.
- Added functions to write unit-tests for rollmelette applications.
- Added Ether wallet.
- Added ERC20 wallet.
- Added application examples.
- Added integration tests with NoNodo.
